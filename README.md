# Codex Skills

个人维护的 Codex Skills 集合。

## Skills

| Skill | 说明 |
|---|---|
| [`codebase-study-coach`](./codebase-study-coach/) | 以可验证的步骤带用户沿真实调用链渐进研读代码库，并持久化学习进度。 |
| [`hermesagent-publish-notes`](./hermesagent-publish-notes/) | 将已确认的学习笔记或获批博文通过 Bearer 认证提交到 HermesAgent。 |

将所需 Skill 目录复制到 `~/.codex/skills/` 后重启 Codex，即可通过 `$codebase-study-coach` 使用该 Skill。
